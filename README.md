## 動作環境
* Windows 10 64Bit
* Microsoft Office 2016 32Bit
* .Net Framework 2.0.50727
* Google chrome


## インストール方法

|&nbsp;|&nbsp;|
|---|---|
|1. ExcelなどのOffice関連をすべて終了させてください|&nbsp;|
|2. WebTools_XXX.exe  をダブルクリックし実行する( XXXはバージョン番号)|&nbsp;|
|3. .Net Framework 2.0がインストールされていない場合は、インストールに移行します。<br>再起動後、インストールをやり直してください|<img src="Docs/Installer-Step00.png" width="80%">|
|4. セットアップウィザードに従い続行|<img src="Docs/Installer-Step01.png" width="80%">|
|5. 利用規約に同意<br>同意できない場合は、キャンセルで終了させてください|<img src="Docs/Installer-Step02.png" width="80%">|
|6. インストールするコンポーネントを選択|<img src="Docs/Installer-Step03.png" width="80%">|
|7. 処理完了まで待機|<img src="Docs/Installer-Step04.png" width="80%">|
|8. インストール完了|<img src="Docs/Installer-Step05.png" width="80%">|

### キャプチャ処理実行方法

キャプチャ対象URLは、シート『WebCaptureList』に記載  
**A列**   サイトURL  　**B列**   出力するシート名  

<img src="Docs/Manual01.png" width="80%">

リストの準備ができれば、リボンメニューのキャプチャボタン <img src="Docs/キャプチャボタン.png"> をクリックすると処理が開始します。
